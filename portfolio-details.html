IOT FINGERPRINT RECOGNITION SYSTEM
November 2024
Capstone Project

ENHANCING HOME SECURITY: DEVELOPING AN IOT FINGERPRINT RECOGNITION SYSTEM FOR DOOR ACCESS CONTROL

lulu.com

Development of a fingerprint-based access control system using Arduino uno and sensors

This project focuses on developing an IoT-based fingerprint recognition system designed to enhance home security by controlling access to residences through biometric authentication. By integrating fingerprint and facial recognition technologies with IoT devices and an Android mobile application, the system allows homeowners to conveniently manage and monitor who enters their homes. The project involves designing and building the necessary hardware, developing a mobile app for remote control and user management, and implementing security measures to safeguard data and prevent unauthorized access. Through testing, the system’s performance, reliability, and user-friendliness are evaluated, providing a modern, secure, and efficient solution that addresses traditional security challenges while boosting homeowners’ confidence in the safety and convenience of their households.

During the development of the IoT-based fingerprint and facial recognition door lock system, challenges included ensuring accurate biometric recognition, maintaining secure data storage, and providing reliable mobile app connectivity. The team also had to address potential sensor errors and create alternative access options to guarantee consistent and secure home entry for users.

To overcome these challenges, the team implemented robust encryption to protect sensitive data, incorporated backup access options via the mobile app and facial recognition, and conducted thorough testing to enhance system accuracy, reliability, and overall user convenience.

  • Facial Recognition
  • Remote Access
